AIOS

An AI-powered operating system for agentic teams.

Structured agent workspaces + a shared collective brain. MIT-licensed, self-hosted. You decide what leaves your machine.

Open source MIT licensed 2026
01 · The problem

Your organization already has a brain.

It's scattered across Slack threads, stale docs, meeting notes, and the person who left six months ago. In agentic teams, context bloats per person, tools entangle, and speed in the wrong direction is waste.

AIOS is the missing layer that makes collective intelligence queryable, actionable, and alive.

02 · Why now

The coordination tax is crushing organizations.

  • 01 The unit of intelligent action is shifting from individuals to human-agent ensembles.
  • 02 Institutions were designed for solo human actors, not hybrid teams of people and agents.
  • 03 Most AI tools are personal productivity. Nobody has built the coordination layer.
  • 04 Repeated meetings, lost decisions, repeated mistakes — the tax compounds.

03 · Principles

Four design commitments.

P1

Modular

Each organ system is independently deployable. Adopt what you need.

P2

Open source

MIT-licensed and self-hosted. Clone per person, run the brain on your Postgres.

P3

Permissioned

Every action is policy-governed. Teams define autonomy vs. human approval.

P4

Privacy-first architecture

Individual data stays personal. The brain receives only what members push.

04 · Architecture

Individual agents → collective brain → surfaces → actions.

05 · The system

A workspace for individuals, a brain for the team.

Your workspace

Numbered folder spine, governance, validators, and harnesses — one per person.

Shared memory

Tier-tagged pushes to one brain. Query in plain English across team context.

Nothing leaks

Private → team → external. Admin-tier rejected at the API boundary.

06 · Runtime-agnostic

One interface for all your agentic work.

Bring your own agent. Extend with MCP. Drive everything from a GUI — no terminal required.

07 · How it flows

A continuous loop, not a pipeline.

Work surfaces from every workspace, enriches the shared brain, and insights return to each individual — tightening with every cycle. Share a skill to the team repo and everyone adopts it.

08 · Vision
"What Claude Code did for individual agent productivity, AIOS does for team agentic productivity."

Every context push makes the system smarter; every query pulls the right signal back. Agents take on execution; humans run orchestration.
