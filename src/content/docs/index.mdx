## Private workspaces. Verified work. Shared brain.

<CardGrid stagger>
  <Card title="AIOS Workspace" icon="laptop">
    One per person. A numbered folder spine, governance conventions, validators, and multi-agent harnesses that produce grounded work. You control what leaves your machine.
  </Card>
  <Card title="Team Brain" icon="puzzle">
    One per team. Receives tier-tagged pushes from every workspace. Surfaces shared tasks, decisions, and deliverables. Answers natural-language questions across the work people chose to promote.
  </Card>
  <Card title="Promotion before sharing" icon="document">
    Everything carries an audience tier — private, team, or external. Content is promoted deliberately through the workspace spine. Admin-tier work stays local.
  </Card>
  <Card title="Open source" icon="open-book">
    Open source and built to be run by any team. Self-host the brain, clone the workspace per person, and keep the operating model in infrastructure you control.
  </Card>
