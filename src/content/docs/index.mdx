## Two repos. One system.

<CardGrid stagger>
  <Card title="AIOS Workspace" icon="laptop">
    One per person. A clean folder spine, governance conventions, validators, and multi-agent harnesses that do real operational work — meeting transcripts into decisions, weekly synthesis, scope-creep detection. You control what leaves your machine.
  </Card>
  <Card title="Team Brain" icon="puzzle">
    One per team. Receives tier-tagged pushes from every workspace. Surfaces shared tasks, decisions, and deliverables. Answers natural-language questions across the team's collective memory.
  </Card>
  <Card title="Context-first design" icon="document">
    Everything carries an audience tier — private, team, or external. Content is promoted deliberately through the workspace spine. Nothing leaks without an explicit push.
  </Card>
  <Card title="Open source" icon="open-book">
    Open source under MIT, built to be run by any team. Self-host the brain; clone the workspace per person. No cloud, no lock-in.
  </Card>
</CardGrid>
